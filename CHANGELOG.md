# Changelog

All notable changes to VoicePi are tracked in versioned release notes under `docs/changelogs/`.

- Latest release notes are published on GitHub Releases: https://github.com/pi-dal/VoicePi/releases
- Repository copies of release notes live at `docs/changelogs/v<version>.md`
- New release notes should be created from `docs/changelogs/TEMPLATE.md`

## Versions

- [v1.5.0](docs/changelogs/v1.5.0.md) — Fallback delivery, automatic prompt bindings, and dictionary learning
- [v1.4.0](docs/changelogs/v1.4.0.md) — Realtime Aliyun/Volcengine ASR and smoother live transcript streaming
- [v1.3.2](docs/changelogs/v1.3.2.md) — Packaged app startup crash fix
- [v1.3.1](docs/changelogs/v1.3.1.md) — Packaging and distribution note update
- [v1.3.0](docs/changelogs/v1.3.0.md) — In-app update support and prompt workspace system
- [v1.2.0](docs/changelogs/v1.2.0.md) — GitHub release update checker
- [v1.1.2](docs/changelogs/v1.1.2.md) — Floating capsule UI and text segmentation
- [v1.1.1](docs/changelogs/v1.1.1.md) — App icon refresh
- [v1.1.0](docs/changelogs/v1.1.0.md) — Provider-aware post-processing and translation
- [v1.0.0](docs/changelogs/v1.0.0.md) — Initial public release
