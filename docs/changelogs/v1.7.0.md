# VoicePi v1.7.0

## Highlights
This release makes realtime transcription start listening sooner, falls back to the standard batch path when realtime never becomes ready, and tightens dictionary learning so accepted corrections are captured more reliably without learning unrelated replacements.

## Added
- Realtime ASR now buffers audio before the live connection is ready so your opening words are preserved instead of being missed during startup.

## Changed
- Realtime ASR sessions now fall back to the existing batch transcription flow when realtime streaming cannot become usable.
- Dictionary learning now keeps tracking accepted edits within the same injection session so repeated corrections can be learned without restarting the app.

## Fixed
- Clearing injected text now stops the active dictionary learning session instead of learning from later placeholder or unrelated text.
- Dictionary suggestion extraction now ignores unrelated replacement words instead of saving bad corrections.
- The changelog site now shows the selected release detail in a simpler inline preview flow.

## Breaking Changes
- None
