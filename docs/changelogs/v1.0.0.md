# VoicePi v1.0.0

## Highlights
First public release of VoicePi, a macOS 14+ menu-bar voice input app.

## Added
- Menu-bar only app (`LSUIElement`) for minimal UI footprint
- Apple Speech local transcription support
- Remote OpenAI-compatible ASR support with configurable API Base URL, API Key, and Model
- Real-time floating capsule overlay with live waveform visualization
- Clipboard-based paste injection with automatic clipboard restore
- Temporary CJK input method switch to ASCII before paste, then restore
- Configurable recognition language (Simplified Chinese, Traditional Chinese, English, Japanese, Korean)
- Optional LLM refinement for conservative transcript correction
- SwiftPM build system with Makefile helpers
- Ad-hoc signed `.app` bundle output
- Homebrew cask installation path (`brew install --cask pi-dal/voicepi/voicepi`)
- GitHub Actions CI workflow
