# VoicePi v1.7.1

## Highlights
- Fixed paste insertion reliability when inserting reviewed text from the result panel.
- Tightened dictionary learning so adjacent existing text does not get learned as part of a new term.

## Added
- None

## Changed
- Tightened dictionary suggestion extraction around prefixed and mixed-script text so VoicePi only learns the intended replacement term.

## Fixed
- Restored reviewed-text insertion by reusing the originally captured input target and frontmost app instead of the VoicePi review panel.
- Prevented dictionary suggestions from learning existing prefix text such as `M你好`, `你好z`, or other surrounding content merged into the corrected term.
- Preserved normal dictionary learning for valid replacements such as `postgre` to `PostgreSQL` and `cloud flare` to `Cloudflare`.

## Breaking Changes
- None
