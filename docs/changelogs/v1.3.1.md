# VoicePi v1.3.1

## Highlights
This patch release fixes a packaging regression that could make the exported app quit immediately on launch because bundled prompt-library resources were missing from the packaged `.app`.

## Added
- None

## Changed
- Updated the README distribution notes for ad-hoc signed test builds, including the expected manual-open flow and a reminder that VoicePi launches as a menu-bar app without a Dock icon.

## Fixed
- Copied Swift Package Manager resource bundles into packaged app exports so release builds include the bundled prompt library and no longer crash at startup.

## Breaking Changes
- None
