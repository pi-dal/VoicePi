# VoicePi v1.4.0

## Highlights
This release upgrades remote realtime dictation with Aliyun and Volcengine streaming ASR, and makes live transcript updates feel continuous instead of chunk-by-chunk jumps.

## Added
- Aliyun DashScope realtime ASR backend via WebSocket streaming.
- Volcengine realtime ASR backend via WebSocket streaming.

## Changed
- Aliyun and Volcengine live partial transcripts are now merged into a running cumulative transcript in the floating overlay.
- Remote ASR prompt handling now always applies VoicePi built-in ASR bias rules first, then appends user-provided prompt text as additional hints.

## Fixed
- Reduced jittery live transcript behavior where the floating overlay previously updated one chunk at a time without reflecting prior context.

## Breaking Changes
- None
