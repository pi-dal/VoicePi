# VoicePi v1.8.0

## Highlights
- Added a new History section in Settings so you can review recent transcript activity and usage trends without leaving VoicePi.
- Expanded stored history metrics to include sessions, character count, word count, and recording duration, with interactive charts for time-based analysis.
- Improved the floating panel so refining banners stay compact for short text but can grow when longer transcript content needs more room.

## Added
- Added a dedicated History section in Settings with saved transcript history stored under VoicePi application support.
- Added usage dashboard views for sessions, characters, words, and recording duration with timeline and weekday-by-hour visualizations.
- Added tooltip and storage tests that cover history persistence and the new History section behavior.

## Changed
- Renamed the Settings Dictionary section to Library to better reflect dictionary and related content management.
- Refactored floating panel sizing support so banner width limits respond to the current panel phase instead of one shared constant.

## Fixed
- Fixed refining banner sizing so longer transcript content can expand beyond the compact width used for short status text.

## Breaking Changes
- None
