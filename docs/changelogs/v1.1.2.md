# VoicePi v1.1.2

## Highlights
Improves the floating capsule UI with a fade mask for overflowing transcript text, and enhances text segmentation for longer dictations.

## Added
- Leading-edge fade mask for overflowing transcript text in the floating capsule to indicate more content is available
- Text segmentation for long translations to prevent chunking artifacts
- Guided permission handoff flow with custom pre-prompts to reduce friction when granting permissions

## Changed
- Refactored permission flow for better user guidance
