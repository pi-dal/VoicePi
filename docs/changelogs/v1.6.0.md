# VoicePi v1.6.0

## Highlights
This release adds a full external processor refinement workflow with dedicated processor profiles, a processor shortcut, and a centered review panel so you can inspect refined output before VoicePi inserts it.

## Added
- A dedicated `External Processors` settings section for creating, enabling, and managing saved processor profiles.
- A processor shortcut that can trigger the external refinement flow directly without changing your standard recording shortcut.
- A centered result review panel with `Insert`, `Copy`, `Retry`, and `Dismiss` actions for external processor output.

## Changed
- External processor refinement now uses a profile-driven workflow that persists processor settings, prompt handling, and command arguments across launches.
- Alma CLI support now ships as the first external processor backend in the refined processor workflow.

## Fixed
- External processor failures no longer silently fall back to injecting the raw transcript into the current target.
- Processor and standard shortcut handling are now better isolated so the dedicated processor path behaves consistently in the menu bar and settings UI.

## Breaking Changes
- None
