# VoicePi v1.10.0

## Highlights
- Added a configurable cancel shortcut so you can abort an active capture or processing run without finishing it.
- Improved External Processor source capture so VoicePi keeps quoted webpage, PDF, and copied-selection context more reliably.
- Restored paste injection reliability after the recent timing regression and expanded benchmark coverage for that path.

## Added
- Added a `Cancel Shortcut` setting in `Settings -> Home`, with `Control + .` as the default binding.
- Added bare `Esc` support as an optional cancel shortcut when both Accessibility and Input Monitoring are granted.

## Changed
- Changed External Processor refinement to keep selected source text across normal refinement runs, selected-text review reruns, and non-editable webpage or PDF selections.
- Changed source capture recovery to fall back to a temporary copy flow when Accessibility APIs do not expose the highlighted text directly.

## Fixed
- Fixed paste injection so clipboard-backed insertion stays reliable without restoring stale clipboard content over a newer user copy.
- Fixed External Processor prompt assembly so missing Accessibility selection metadata no longer drops the quoted Source block as often.

## Breaking Changes
- None
