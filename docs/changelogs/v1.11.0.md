# VoicePi v1.11.0

## Highlights
- Refreshed the Settings window with a warmer light theme, richer dark theme, and cleaner card and navigation styling while preserving the compact menu-bar workflow.
- Added a new History usage visualization with timeline and heatmap views so you can understand recording activity at a glance.
- Improved the Dictionary management experience with collection navigation, search, and clearer term-detail presentation.

## Added
- Added History usage visualizations, including heatmap and line-chart views for recent activity trends.
- Added History filters for all dates, today, last 7 days, and last 30 days, plus sorting and pagination controls for longer histories.
- Added Dictionary collection navigation and search so larger libraries are easier to browse.

## Changed
- Changed the Settings window visual system with updated surfaces, navigation treatment, and button styling across light and dark appearances.
- Changed the Dictionary layout to use a clearer sidebar-and-detail structure with stronger headings and scrollable content areas.
- Changed prompt template editing to open in a dedicated sheet-based editor from the refreshed Settings experience.

## Fixed
- Fixed several Settings layout regressions uncovered during the visual refresh so the existing compact window height and scrollable sections remain usable.
- Fixed Dictionary and History library presentation edge cases that made dense content harder to scan.

## Breaking Changes
- None
